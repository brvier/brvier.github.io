ForRunners

ForRunners is an personnal tracking app for your running which respect your privacy.

ForRunners use the GPS of your phone to track your run.During a run speech synthesis can announce distance, speed, heart rate. And after your run you can see various information to help you improve your next running with nice looking graphics and statistics. ForRunners can also connect to a Bluetooth Smart Heart Rate Sensors if you have one. ForRunners can also help you if you want to do interval running, by announcing interval via speech synthesis.

Respect your privacy

  • Don't share any informations with third party server
  • Use OpenStreetMap
  • Don t retain your datas, you can export your run to GPXs files

Use low ressources

  • Use white text on black screen when running, less power used on Amoled screen
  • Don t require network connection while running

Nice Statistics and charts

  • See your runs on a map
  • Your best runs are registered by length of your run
  • Displays charts with speed/altitude, altitude/hearthrate, and hearthrate zone

Modern look and feel

  • Use Material Design
  • Easy to use

BitPurse

A nice looking Bitcoin Client for MeeGo and Harmattan.

BitPurse allow you to manage your bitcoin address, view your balance, and emit transaction directly from your mobile device.

Get it on OpenRepos.net

Secure

  • Your main wallet password will be saved on your phone however you can optionally set a second password required before you can make payments. This means even if your phone gets stolen your funds will still be safe!
  • 100% Free and Open source : Use open source technology Python, PySide, Qt, PyCrypto, PyPackager, Git, Python ECDSA module, and Python PBKDF2 module. The source can be found on my github repository and are licensed under GPLv3

No Blockchain download

  • BitPurse uses Blockchain.info api saving valuable space and bandwidth.

RulesPasswords

A password hasher.

A mobile application for Blackberry 10, Android, iPhone or other plateforms to generate a password from a hash made with the domain name of a site web and a master password. RulesPasswords is an hybrid application made with Cordova and OnsenUI.

Features

  • Compatible with PasswordMaker extension for FirefoxOS or Chrome
  • 100% Free and Open source : Use open source technology Html5, OnsenUI, Cordova, PasswordMaker javascript library. The source can be found on my github repository and are licensed under GPLv3

Privacy Policy

  • The application did't require any permission except on Blackberry to read the type of device.
  • The application never communicate outside, and none information are collected.
